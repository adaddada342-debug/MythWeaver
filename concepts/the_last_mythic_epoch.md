# The Last Mythic Epoch

## Pack name

**The Last Mythic Epoch**

## Core concept

A mythic fantasy survival sandbox where the world feels like it survived several collapsed ages of civilization. The player explores beautiful dangerous terrain, ruined kingdoms, caves, villages, dungeons, relics, restrained magic, bosses, settlement systems, building, farming, cooking, and late-game mysteries. It should feel huge, ancient, and alive, but **stable and playable first** — not random mod soup.

## Target experience

- Strong exploration and ancient-world tone
- Restrained structure/worldgen stack (no competing overhauls)
- Coherent UI/recipe layer (EMI; Jade only — no WTHIT)
- Single simple storage route
- Performance-first Fabric stack
- Phase 1: **no** Create addons, Valkyrien Skies/Eureka, Ad Astra, horror stalker mods, duplicate Cave Dweller / Create-addon / food-progression variants

## Technical targets

| Item | Target |
|------|--------|
| Minecraft | 1.20.1 |
| Loader | Fabric |
| Phase 1 mod count | **80–120 mods including dependencies** (hard cap **150** for this pass) |
| Stretch | 150 only after a first version launches and is verified |

Prefer Modrinth. Prefer **launch-first** over mod count: a stable ~95-mod pack beats a broken 300-mod list.

## Design pillars

1. **Performance foundation** — Sodium family, Lithium, FerriteCore, ModernFix, entity culling; no acceleratedrendering.
2. **Terrain / exploration (conservative)** — One primary overworld expansion (e.g. Terralith) + compatible add-ons only when verified; limited YUNG structures; towns/villages flavor without overload.
3. **Building / decoration** — Supplementaries, Moonlight, chipped, handcrafted, rechiseled, curated Macaw set.
4. **Farming / cozy survival** — Farmer's Delight Refabricated + minimal verified addons; Nature's Compass; one backpack solution.
5. **Fantasy / combat (light Phase 1)** — Better Combat, Simply Swords, Spell Engine + Wizards where compatible; mythic mounts; no huge unstable magic stacks.
6. **Utility / travel** — Waystones; optional voice chat / claims only when source-resolve and artifact validation are clean.
7. **UI / QoL** — EMI, AppleSkin, Jade, Xaero map pair, shulker tooltip, mouse tweaks; Reese's options / Sodium Extra if compatible.

## Player fantasy

Ancient realms, mystery in the landscape, and progression that rewards patience — **Prominence/Better MC exploration scale in ambition, MythWeaver-verified curation in practice** for Phase 1.

## Non-goals (Phase 1)

- Maximizing mod count
- Multiple huge worldgen stacks
- Jade + WTHIT together
- Multiple storage ecosystems
- Craftable-vanilla-resource filler, hack/autofish/bedrock miner/fakename/nofabric/neko-hacks style mods
- Alpha/beta unless essential and justified
- Forcing past final artifact validation failures
