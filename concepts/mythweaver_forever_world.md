# MythWeaver Forever World (concept stub)

Long-term **vanilla-friendly** industrial megabase pack: **Minecraft 1.20.1 Fabric**, **Create** backbone, catalog farming, physical logistics, restrained worldgen, optional environmental lore.

**Authoritative design + mod list + matrices** live under:

`output/generated/mythweaver-forever-world/`

Use MythWeaver:

```powershell
python -m mythweaver.cli.main design-pack concepts/mythweaver_forever_world.md --output-dir output/generated/mythweaver-forever-world --name "MythWeaver Forever World"
```

…then merge any new `pack_design.json` fields with the checked-in design if you re-run design automation.
