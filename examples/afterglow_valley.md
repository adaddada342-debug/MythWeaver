# Afterglow Valley

Afterglow Valley is a cozy Fabric 1.20.1 exploration and survival pack about settling into a beautiful world. The fantasy is simple: find a golden valley, build a cottage, grow food, decorate warm interiors, restore villages, follow rivers and roads, discover gentle ruins, and slowly turn a home into a lived-in settlement.

The pack intentionally avoids horror, hardcore survival realism, guns, explosives, large tech chains, boss-heavy RPG progression, and meme content. Progression comes from comfort and beauty: better food, better storage, better travel, better building palettes, better maps, and a wider exploration range.

The selected stack prioritizes performance, scenic terrain, improved villages, light structures, wildlife, ambience, sound, cozy building blocks, furniture, farming/cooking, maps, backpacks, base storage, and measured fast travel. Shaders are not required for the core experience.

Stability rules:

- Required dependency closure must pass before export.
- Source resolution must use verified automated sources where possible.
- CurseForge may only be used through the official API when configured.
- Runtime stability must be proven by MythWeaver or reported honestly as manual_required.
- Optional unstable mods should be removed or replaced without losing the cozy exploration identity.
