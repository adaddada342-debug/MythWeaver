# Peacekeeper / Worldbreaker

Peacekeeper / Worldbreaker is built around a deliberate contrast: a beautiful, cozy survival world
that can also become a playground for opt-in destruction. The default experience is relaxed
worldbuilding: scenic terrain, improved villages, wildlife, cooking, farming, decoration, roads,
bridges, windmills, and long-term settlement projects.

The Worldbreaker route is present but narrow on purpose. The pack uses one focused gun system,
one spell/power route, Better Combat plus Simply Swords for satisfying combat escalation, and
Create Big Cannons as the late-game destructive bridge. Create is included because it supports both
sides of the fantasy: peaceful windmills and workshops early, siege-scale machinery later.

## Intended Player Paths

- Peacekeeper: build a home, improve villages, cook better meals, decorate interiors, connect
  settlements, and use power defensively.
- Worldbreaker: craft ammunition, build dangerous machinery, learn spells, collect stronger weapons,
  and use destructive force intentionally.
- Small SMP: enable claims/protection and agree on where destructive weapons are allowed.

## Balance Notes

- Guns should stay resource-bound through ammo crafting.
- Create Big Cannons should be treated as late-game infrastructure, not first-night chaos.
- Claims/protection are recommended for SMP play.
- Avoid using destructive systems near peaceful builds unless the server intentionally wants that
  risk.
- Human taste review should test whether Wizards and Simply Swords feel too strong without extra
  gating.

## Suggested Config Profiles

- Peacekeeper profile: claims enabled, destructive testing areas encouraged, heavy grief tools
  restricted near settlements.
- Worldbreaker profile: claims optional, cannons and destructive play allowed by server agreement.
- Solo cinematic profile: all systems enabled, but destructive tools used as personal escalation
  goals rather than mandatory progression.
